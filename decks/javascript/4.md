# What is the difference between the function declarations below?

```javascript
const foo = function() {
    // Some code
};
```

```javascript
function bar() {
    // Some code
};
```

%

- Function `foo` is defined at runtime
- Function `bar` is defined at parse time

_Note: Due to hoisting, this might not actually matter?_
